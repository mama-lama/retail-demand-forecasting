# Stockly

Веб-приложение для прогнозирования спроса на товары розничного магазина на основе исторических данных о движении товаров.  
Проект предназначен для анализа продаж и формирования прогнозов спроса, которые могут использоваться для оптимизации закупок и ценообразования.

## Концепт проекта

Проект представляет собой веб-приложение для прогнозирования спроса на товары розничного магазина на основе данных о движении товаров за **4 года**.

Источником данных является **GraphQL API**, через который получаются следующие данные:

- приход товаров  
- продажи  
- списания  
- результаты инвентаризации  
- закупочная цена  
- цена продажи  
- дата операции  

На основе этих данных система строит прогноз продаж по каждому товару на следующие горизонты:

- **1 день**
- **7 дней**
- **14 дней**
- **30 дней**

Результаты прогнозирования отображаются пользователю в виде:

- таблиц  
- графиков
